# Django firm manager (Рабочее название)

Небольшая CRM система для работы в небольшом предприятии.
## Реализовано:
* Работа с клиентами:
  * Создание лидов;
  * Создание контрагентов.
* Создание документов (договор, заказ, спецификация) и генерация файлов в формате docx для отправки клиентам
* Небольшие нароботки складского учета:
  * приходы продукции;
  * расходы продукции; 
  * брони продукции по заказам.

## Что необходимо реализовать
* Тесты для имеющегося функционала
* Прохождение лидов и контрагентов по статусам
* Задачи для сотрудников
* Недостающие документы для ведения работы с клиентами
